# Dashboard de Análise de Vendas de Carros

## Descrição do Projeto
Este projeto desenvolve um aplicativo web interativo usando Streamlit para análise exploratória de dados de anúncios de vendas de carros. O dashboard permite visualizar padrões nos dados através de histogramas e gráficos de dispersão.

## Funcionalidades
- 📊 Histograma interativo da quilometragem dos veículos
- 📈 Gráfico de dispersão mostrando relação entre quilometragem e preço
- ✅ Interface com caixas de seleção para controle dos gráficos
- 🌐 Aplicativo web responsivo e interativo

## Tecnologias Utilizadas
- Python
- Streamlit
- Pandas
- Plotly Express

## Como Executar Localmente
1. Clone este repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute o aplicativo: `streamlit run app.py`
